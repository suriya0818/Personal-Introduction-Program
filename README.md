Personal Introduction Program:

What I Learned:

Through this project, I learned the basics of Python programming. I learned how to take input from users, store information in variables, and display output using print statements. I also learned how to run a Python program and test it with different inputs. This project helped me understand how Python can be used to create simple interactive applications.

Project Overview:

The purpose of this project is to create a Personal Introduction Program using Python. The program collects basic information from the user such as their name, age, nationality, and hobby. After collecting the information, it displays a friendly introduction message based on the user's input.

Setup Instructions:

1. Install Python on your computer.
2. Create and save the program as personal_intro.py.
3. Open Command Prompt or Terminal.
4. Navigate to the folder where the file is saved.
5. Run the program using the following command:

python personal_intro.py

Code Structure:

Technical Details:

This project uses the following Python concepts:

-Variables to store user information
-input() function to collect user input
-print() function to display output
-f-strings for formatting text and messages

Program Flow:

User Input -> Store Information -> Generate Introduction -> Display Output
